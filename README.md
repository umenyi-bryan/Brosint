# BROsint

**Author:** Chinedu  
**Version:** 1.0.0  
**Description:**  
BROsint is an advanced Open Source Intelligence (OSINT) framework designed to gather, analyze, and visualize publicly available information — including email footprints, social links, and data patterns — for ethical and educational use.

---

### Features
- Email and phone metadata scanning  
- Social account footprint linking  
- Smart hypothesis generation  
- Neon CLI interface with Rich animation  

---

### Installation

```bash
git clone https://github.com/ChineduDev/BROsint.git
cd BROsint
python3 -m venv venv
